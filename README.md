# Course Finder App 

A simple and responsive React application that helps users browse and search for courses easily.  
The app also displays random inspirational quotes using the **Kanye REST API**.

# Features

- Browse a list of courses
- Search courses by name
- Random quote generator with refresh option


# Tech Stack

- React
- JavaScript
- Vite
- CSS

## External API Used

### Kanye REST API
- **Endpoint:** https://api.kanye.rest/
- **Usage:** Fetches a random Kanye West quote
